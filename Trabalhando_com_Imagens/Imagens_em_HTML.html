Imagens em HTML

Imagem Interna

Para utilizar imagens em um site, é necessário utilizar a tag img. Ao apertar ctrl + espaço dentro do parâmetro src, irá surgir um menu mostrando as pastas do projeto, logo após isso, deve-se escolher a pasta que armazena a imagem desejada.

Imagem Externa

Também é possível utilizar imagens que se encontram fora do próprio site, basta copiar o link da imagem e incluí-lo no código fonte.
